Image-to-Video Converter

Convert your images into a video slideshow effortlessly!

How It Works

  1. Download the tool from this repository.
  2. Place your images in the images/ folder.
  3. Run the script using Python:
    4. python script.py
  4. Your video will be saved as output_video.mp4.

Features

  • Converts images to video using OpenCV.
  • Supports JPG and PNG formats.
  • Adjustable frames per second (FPS).
